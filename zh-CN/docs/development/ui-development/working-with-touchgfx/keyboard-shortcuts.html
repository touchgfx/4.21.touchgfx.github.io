键盘快捷键

下面列出的是TouchGFX支持的键盘快捷键完整列表，以提高工作效率。

TouchGFX Designer功能​

文件管理​

CTRL + N显示大厅
CTRL + O从文件资源管理器打开项目
CTRL + S保存当前项目

控件菜单​

A显示控件菜单（如果已打开，则显示焦点搜索文本框）
ENTER插入当前突出显示的控件并关闭控件菜单
ESC关闭控件菜单

画布​

F2更改控件名称
DEL删除选定的控件
CTRL + C复制
CTRL + V粘贴
CTRL + Z撤消
CTRL + Y重做
CTRL + F向前移动选定的控件
CTRL + B向后移动选定的控件
CTRL + A选择所有控件
← / ↑ / → / ↓将选定的控件移动1像素
CTRL + ← / ↑ / → / ↓将选定的控件移动10像素
MOUSE WHEEL向上滚动/向下滚动
SHIFT + MOUSE WHEEL向左滚动/向右滚动
CTRL + MOUSE WHEEL UP / '+'放大
CTRL + MOUSE WHEEL DOWN / '-'缩小
CTRL + 1缩放到25%
CTRL + 2缩放到50%
CTRL + 3缩放到75%
CTRL + 4缩放到100%
CTRL + 0重置缩放比例
MOUSE DRAG选择控件
CTRL + MOUSE DRAG平移画布

调试​

F4生成代码
F5运行模拟器
F6运行目标
CTRL + L显示/隐藏详细日志记录

TouchGFX Simulator功能​

模拟器​

F1启用/禁用指针坐标的显示以及该坐标上像素的RGB颜色（十六进制）。
F2启用/禁用突出显示无效区域
F3截屏并将图片置于screenshots（截屏）文件夹中
CTRL + F3对接下来的50帧进行截屏并将图片置于screenshots（截屏）文件夹中
SHIFT + F3截屏并将图片置于剪贴板中
F4如果使用模拟器皮肤 - 启用/禁用模拟器皮肤。
如果未 使用模拟器皮肤 - 启用/禁用窗口边框
F5通过调用FrontendApplication::changeToStartScreen()将应用程序直接发送回启动屏幕.
F9通过阻止向应用程序发送时钟信号拍来暂停模拟器。 再次按下F9可恢复正常执行。
F10当（按下F9）模拟器暂停后，可以通过按下F10发送一个时钟信号到应用程序，从而对应用程序进行“单步操作”。
ESC关闭模拟器
