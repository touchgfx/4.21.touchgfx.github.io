入门

在开始使用TouchGFX前，需要安装最新版的TouchGFX，这将在安装部分中介绍。

完成该操作后，您就可以开始第一个TouchGFX项目。 强烈建议您阅读本文的关键章节，以熟悉最基本的TouchGFX概念。 最基本的章节为开发简介，此部分将为您介绍完整的TouchGFX项目所需的软硬件组件，以及创建它们所需的操作及工具。 该章节还介绍了如何使用预制组件开始快速原型设计。

TouchGFX Quick Start​

If you are eager to start experimenting with TouchGFX, trying out an example UI project and maybe running it on an STM32 Evaluation Kit, a wide range of TouchGFX Board Setups together with GUI Examples and Demos exists in TouchGFX Designer.

TouchGFX 板级设置 + 示例 及 Demos​

To create a TouchGFX project based on a TouchGFX Board Setup go ahead and start the TouchGFX Designer.

基于TouchGFX板级设置创建一个应用示例

在TouchGFX主界面，有大量的示例和DEMO可供选择. 您甚至可以将该示例与一系列STM32评估套件的板特定代码相结合，以便立即在硬件上尝试该示例。

如果没有STM32评估套件，您只需在PC上运行TouchGFX应用程序。 如果您希望根据操作指南完成第一步，则可以查看教程章节。 这里将向您展示如何启动并运行项目，最终完成一个相当复杂的UI应用程序。

特定单板的Demos​

除了能够将应用程序模板与UI模板相结合以外，TouchGFX Designer还包括一些针对特定STM32评估套件的现成演示。 These are located in the Demos section under Board Specific Demo.

下一步​

此时，建议您浏览文档并阅读相关章节。 这将使您对文档有一个大致的了解，从而让您知道从哪里查找更多信息，并了解TouchGFX关键概念。

主要章节包括：

  • 基本概念: 本章为本文档的其余章节提供背景知识。 其中将介绍稍后要引用的所有关键图形概念。 根据您当前的知识以及您将进行的TouchGFX开发操作，您可能想先跳过某些章节，并在以后不了解某些细节的情况下回过头来查看这些章节。
  • 研发: 本章为本文档的主要章节。 它解释了TouchGFX项目的结构以及整个项目开发周期中涉及的工作流程和工具。 工作流程中的每个步骤都有自己的章节，并对其进行了详细描述。 您可以在其中找到有关如何使您的应用程序在硬件上运行，以及如何进行UI开发的全部所需知识，包括对TouchGFX Framework中的所有功能和组件的描述。
