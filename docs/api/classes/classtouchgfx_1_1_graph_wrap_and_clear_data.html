GraphWrapAndClearData

touchgfx/widgets/graph/GraphWrapAndClear.hpp

The GraphWrapAndClearData will show new points progressing across the graph. Once the graph is filled, the next point added will cause the graph to be cleared and a new graph will slowly be created as new values are added.
