키보드 단축키

다음은 TouchGFX에서 생산성을 높이기 위해 지원하는 전체 키보드 단축키 목록입니다.

TouchGFX Designer 기능​

파일 관리​

CTRL + N로비 표시
CTRL + O파일 탐색기에서 프로젝트 열기
CTRL + S현재 프로젝트 저장

위젯 메뉴​

A위젯 메뉴 표시(이미 열린 경우 검색창으로 이동)
ENTER현재 강조 표시된 위젯 삽입 후 위젯 메뉴 닫기
ESC위젯 메뉴 닫기

캔버스​

F2위젯 이름 변경
DEL선택한 위젯(들) 삭제
CTRL + C복사
CTRL + V붙여 넣기
CTRL + Z실행 취소
CTRL + Y재실행
CTRL + F선택한 위젯(들)을 앞으로 보내기
CTRL + B선택한 위젯(들)을 뒤로 보내기
CTRL + A모든 위젯 선택
← / ↑ / → / ↓선택한 위젯(들)을 1 픽셀씩 이동
CTRL + ← / ↑ / → / ↓선택한 위젯(들)을 10 픽셀씩 이동
MOUSE WHEEL위로 스크롤 / 아래로 스크롤
SHIFT + MOUSE WHEEL왼쪽으로 스크롤 / 오른쪽으로 스크롤
CTRL + MOUSE WHEEL UP / '+'확대
CTRL + MOUSE WHEEL DOWN / '-'축소
CTRL + 1확대/축소 25%
CTRL + 2확대/축소 50%
CTRL + 3확대/축소 75%
CTRL + 4확대/축소 100%
CTRL + 0확대/축소 리셋
MOUSE DRAG위젯 선택
CTRL + MOUSE DRAG캔버스 좌우 회전

디버깅​

F4코드 생성
F5시뮬레이터 실행
F6대상 실행
CTRL + L상세 로그 표시/숨김

TouchGFX 시뮬레이터 기능​

시뮬레이터​

F1포인터 좌표 표시와 해당 좌표(16진수) 픽셀의 RGB 색상 표시 활성화/비활성화
F2무효화 영역의 강조 표시를 활성화/비활성화
F3스크린샷 생성 후 screenshots 폴더에 해당 이미지 저장
CTRL + F3다음 프레임 50개의 스크린샷 생성 후 screenshots 폴더에 해당 이미지 저장
SHIFT + F3스크린샷을 생성한 후 클립보드에 저장
F4시뮬레이터 스킨을 사용하는 경우 시뮬레이터 스킨을 활성화/비활성화
시뮬레이터 스킨을 사용하지 않는 경우에는 창 경계를 활성화/비활성화
F5FrontendApplication::changeToStartScreen()을 호출하여 애플리케이션을 시작 화면으로 바로 전송
F9애플리케이션으로 실행 이벤트(tick)를 전송하지 못하도록 하여 시뮬레이터를 일시 중지 다시 F9를 누르면 정상 실행이 재개됨
F10F9를 눌러 시뮬레이터가 일시 중지된 상태에서 F10을 누르면 애플리케이션으로 단일 실행 이벤트(tick)를 전송하여 애플리케이션 "싱글 스텝"이 가능함
ESC시뮬레이터 닫기
