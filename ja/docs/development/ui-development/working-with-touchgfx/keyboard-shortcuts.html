Keyboard Shortcuts（キーボード･ショートカット）

ここでは、生産性向上のためにTouchGFXによって提供されているすべてのキーボード･ショートカットのリストを示します。

TouchGFX Designerの機能​

ファイル管理​

Ctrl + NLobbyを表示
Ctrl + Oファイル･エクスプローラからプロジェクトを開く
Ctrl + S現在のプロジェクトを保存

ウィジェット･メニュー​

Aウィジェット･メニューを表示（すでに開いている場合は検索テキストボックスにフォーカス）
Enter現在ハイライトされているウィジェットを挿入し、ウィジェット･メニューを閉じる
ESCウィジェット･メニューを閉じる

キャンバス​

F2ウィジェットの名前を変更
DEL選択したウィジェットを削除
Ctrl + Cコピー
Ctrl + V貼り付け
Ctrl + Z元に戻す
Ctrl + Yやり直し
Ctrl + F選択したウィジェットを前面に移動
Ctrl + B選択したウィジェットを背面に移動
Ctrl + Aすべてのウィジェットを選択
← / ↑ / → / ↓選択したウィジェットを1ピクセル移動
Ctrl + ← / ↑ / → / ↓選択したウィジェットを10ピクセル移動
マウス･ホイール上へスクロール / 下へスクロール
Shift + マウス･ホイール左へスクロール / 右へスクロール
Ctrl + マウス･ホイールを上へ / '+'拡大
Ctrl + マウス･ホイールを下へ / '-'縮小
Ctrl + 125%にズーム
Ctrl + 250%にズーム
Ctrl + 375%にズーム
Ctrl + 4100%にズーム
Ctrl + 0ズームのリセット
マウス･ドラッグウィジェットを選択
Ctrl + マウス･ドラッグキャンバスのパン

デバッグ​

F4コードの生成
F5シミュレータの実行
F6ターゲットの実行
Ctrl + L詳細ログの表示 / 非表示

TouchGFXシミュレータの機能​

シミュレータ​

F1ポインタ座標とその座標のピクセルのRGBカラーの表示（16進数）を有効化 / 無効化
F2invalidateされた領域のハイライトを有効化 / 無効化
F3スクリーンショットを撮り、screenshotsフォルダにその画像を置く
Ctrl + F3次の50フレームのスクリーンショットを撮り、screenshotsフォルダにその画像を置く
Shift + F3スクリーンショットを撮り、クリップボードにその画像を置く
F4シミュレータ･スキンが使用されている場合、そのシミュレータ･スキンを有効化 / 無効化
シミュレータ･スキンが使用されていない場合、ウィンドウ境界を有効化 / 無効化
F5FrontendApplication::changeToStartScreen()を呼び出すことで、アプリケーションを起動画面に直接戻す
F9ティックがアプリケーションに送信されないようにして、シミュレータを一時停止します。 F9キーをもう一度押すと、通常の実行が再開されます。
F10シミュレータの一時停止中（F9キーの押下後）、F10キーを押すことでアプリケーションに1ティックだけ送信し、それによってアプリケーションを"1ステップ"進めることができます。
ESCシミュレータを閉じる
